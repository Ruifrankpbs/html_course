Atributo "src"

Utilizado para referenciar o local da imagem



Atributo "width"

Utilizado para ajustar a largura da imagem



Atributo "height"

Utilizado para ajustar a altura da imagem



Atributo "alt"

Utilizado para incluir a descrição da imagem, usada para tornar a pagina mais acessivel para deficientes visuais, pois os leitores automáticos lerão as informações contidas dentro deste atributo
