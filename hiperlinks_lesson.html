Para inserir hiperlinks em uma página html, precisamos utilizar a Tag "a"



Mas não basta colocar apenas a Tag "a", mas precisamos colocar o atributo "href" onde possibilitará que seja informado o endereço de algum link externo, que pode ser uma outra página html ou endereço para baixar arquivos



Em uma Tag podemos adicionar varios atributos, podemos ainda acrescentar na Tag "a" o Atributo "target" , que permitirá que ao clicar no link que criamos com a Tag "a" que direciona para qualquer endereço colocado dentro do atributo "href", seja aberta uma nova aba ou guia no nosso navegador. A configuração que podemos colocar no atributo "target" para fazer isso é o "_blank"

Mas o atributo href pode conter tanto páginas da web como links, como arquivos dentro do nosso computador ou em servidores locais e em nuvem. Por exemplo dentro do atributo href:"./minha_pasta_local/index.html".Nesse exemplo, estamos acessando um arquivo html que se encontra dentro de uma pasta dentro do nosso computador
