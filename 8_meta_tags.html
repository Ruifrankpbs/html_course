Tag "link" e atributo "rel="icon""

Utilizada para importar uma imagem como icone de aba da nossa página html.

Tag "title"

Utilizada para nomear a aba da nossa página.

Tag "Meta" e atributo "charset="UTF-8""" e "name="viewport"""

A tag "meta" utilizada para configurar nossa pagina html de forma que ela funcione adequadamente independente do nosso browser. o atributo 'charset="UTF-8' é utilizado para resolver problemas de compatibilidade com os caracteres especiais como cedilha, sempre utilizamos ela como boa prática em paginas html.o atributo 'name="viewport"' juntamente com a 'content="width=device-width, initial-scale=1"', configura a tela do dispositivo cujo o usuario está visualizando a página.
