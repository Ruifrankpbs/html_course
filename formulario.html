Forms

Formulários são importantes quando precisamos que o usuario forneça informações. Formulários são basicamente formados pela Tag input. Seguidas dos atributos type, placeholder,required,name.

type

Basicamente informa que tipo de dado será inserido no campo. Pode ser do tipo Texto, E-mail.

placeholder

É o texto que fica dentro da caixa, informando ao usuario o que deve ser digitado naquele campo.

required

É o atributo que faz com que determinado input seja obrigatório para o envio das informações do Formulário.

Button

É a Tag para inserir o botão do nosso formulário. Também possui os atributos type, onde geralmente configuramos como submit. O que significa que iremos submeter os dados ali informados

textarea

É a Tag onde iremos criar uma caixa de texto mais extensa. Onde poderemos informar a quantidade de linhas com o atributo rows, nele colocamos a quantidade de linhas que queremos nessa caixa de texto.

Atributo "type" radios

Ao configurar o atributo type como radios colocamos um seletor em forma de circulo. E para que possamos selecionar apenas um, pois por padrão vem para selecionar vários, devemos settar com o atributo name com a mesma classificação, que pode ser qualquer nome que você desejar, colocando com o mesmo o name você informa o navegador que todas a opções são do mesma classificação. Nesse exemplo abaixo temos name configurado como "genero", assim podemos selecionar apenas um.

Tag "select" e "option"

Com essas Tags podemos criar opções do tipo lista oculta, quando clicarmos ela mostrará a lista de opções disponiveis. Nesse exemplo abaixo temos o assunto do formulario de contato como lista oculta.
